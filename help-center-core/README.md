# Help center core
Service responsible for issues and comments management, sending notification about new comments via RabbitMQ.
## Technologies
- Java 17
- Spring Framework 5
  - Spring Boot
  - Spring Security  
  - Spring Data
  - Spring MVC 
  - Spring Cloud
  - Spring AMQP
- OpenFeign
- RabbitMQ
- JUnit 5
- Maven
- Hibernate
- PostgreSQL, H2
- Lombok 
- MapStruct
